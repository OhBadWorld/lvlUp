# 中途出现的错误
1. eslint 报 错
(```)
  Home.vue
  1:1  error  Parsing error: Unexpected token <
  我Home.vue 的代码是最简的，它也能给我报错
  <template>
    <div>
      123
    </div>
  </template>
(```)
2. eslint 报 import 导入的问题
.eslintrc.js 中加入
(```)
parserOptions: {
  sourceType: 'module'
},
(```)
3. eslint 又报 当ecmaVersion <2015时，不支持sourceType'module' 问题
更改上面的 .eslintrc.js 为：
(```)
parserOptions: {
  "ecmaVersion": 2015,
  sourceType: 'module'
},
(```)
